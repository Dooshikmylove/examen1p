Comandos Git

○ pwd

Ruta actual (on toy?)

○ ls

Listar contenido de directorio

○ git init

Inicia o crea repositorio

○ git status

Estado de repositorio

○ git add

Agregar archivos al staging area

○ git config

Configurar usuario o email

○ git branch

Cambiar de nombre la ama principal

○ git log

Ver la rama completa

○ git push

Subir repositorio local o remoto

○ git pull

Bajar los cambios
